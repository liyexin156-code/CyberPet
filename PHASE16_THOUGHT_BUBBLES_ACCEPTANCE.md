# PRD 第 16 节验收说明：思考泡泡

## 运行

```bash
DEVELOPER_DIR=/Applications/Xcode.app/Contents/Developer CLANG_MODULE_CACHE_PATH=.build/module-cache swift run PetTaskBuddy
```

## 验收点

- 今日未完成条目从宠物头顶上方以独立 pill 泡泡显示，不再使用侧边清单。
- 狗头顶到泡泡群之间有 3 个由小到大的思考圆点。
- 任务泡：圆形勾选框 + 标题；点击圆圈会完成任务，并复用 Phase 1b 喂宠物逻辑。
- 提醒泡：铃铛 + 标题 + 可选提醒时间；点击提醒泡只收起，不喂宠物。
- 泡泡左右错落，最多显示 4 个；更多条目收进顶部 `+N` 泡。
- 泡泡逐个延迟浮现，并有轻微上下浮动。
- 泡泡窗口跟随宠物，靠近屏幕左右边缘时整体收进屏幕内。
- 主面板仍然保留完整添加、编辑、备注和日程管理。
- 今日无条目时不显示；任务全部完成后显示「今天都照顾好啦～」。
- 主面板「今日」里有「思考泡泡常驻显示」开关；关闭后悬停到狗狗才显示。
